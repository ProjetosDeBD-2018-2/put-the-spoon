### O que está incluso:

Ao clonar ou baixar o este projeto, você o encontrará com a seguinte estrutura:

```
x_lbd_free/
├── assets/
|   ├── css/
|   ├── sass/
|   ├── js/
|   ├── fonts/
|   └── img/
├── consultas/
|   ├── consulta1.html
|   ├── consulta2.html
|   ├── consulta3.html
|   ├── consulta4.html
|   └── consulta5.html
├── CHANGELOG.md
├── LICENSE.md
├── README.md
├── index.html
├── glossario.html
└── quem-somos.html
```

### Como rodar o projeto:

É somente necessario abri o  arquivo `index.html` no navegador.
